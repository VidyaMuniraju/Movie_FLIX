My Flix Project

This is an web application that will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Description of the request URL Endpoint HTTP Method Type Parameters in URL Format of data in the body of request Format of the response data
Return a list of all movies to the user /movies GET None None A JSON object holding data about all movies
Return data about a single movie to the user /movies/:title GET Request parameter None A JSON object that holds data - genre, description, director's name, and an image URL of the movie requested
Return data about a genre /movies/genre/:Name GET Request parameter None A JSON object holding data about the requested genre - name and description
Return data about a director /movies/director/:Name GET Request parameter None A JSON object holding data about the director - name, bio, year of birth, year of death(if not empty)
Allow new users to register /users POST None A JSON object hodling data about the new user - username, password, emailID, and date of birth A JSON object holding data about the new user - an userID, username, list of favorite movies(must be empty)
Allow users to update their user info(username) /users/:Username PUT Request parameter A JSON object holding the updated data - username, emailID A JSON object holding data - userID, username, password, emailID, and date of birth which reflects the changes made
Allow users to add a movie to their list of favorites /users/:Username/favoritemovies/:title POST Request parameter A JSON object holding data about the movie to add - name, description, genre, director's name, and an image URL A text message indicating that the movie was added to the favorites list of the user
Allow users to remove a movie from their list of favorites /users/:Username/favoritemovies/:title DELETE Request parameter None A text message indicating that the movie was deleted from their list of favorites
Allow existing users to deregister /users/:Username DELETE Request parameter None A text message indicating that the user has been deregistered
